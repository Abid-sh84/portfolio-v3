# Portfolio v3

A full-stack developer portfolio showcasing projects, skills, and technical insights. Built with React, Next.js, Tailwind CSS, and Framer Motion for a responsive dark-themed user experience.

**Live Site:** [abids.tech](https://abids.tech)

## Quick Start

### Prerequisites
- Node.js 18 or higher
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Abid-sh84/portfolio-v3.git
cd portfolio-v3

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the portfolio.

## Build & Deployment

```bash
# Build for production
npm run build

# Start production server
npm start
```

## Tech Stack

- **Framework:** Next.js 16
- **Styling:** Tailwind CSS 4
- **UI Components:** Radix UI
- **Animations:** Framer Motion
- **Content:** MDX with Next MDX Remote
- **Code Highlighting:** Rehype Highlight
- **Theme Support:** Next Themes
- **Utilities:** Lucide Icons, Clsx

## Project Structure
